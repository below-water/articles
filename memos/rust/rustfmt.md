# rustfmt

## Options

| option  | description                                                                                                   |
| ------- | ------------------------------------------------------------------------------------------------------------- |
| --all   | workspace全体のクレートを対象にする。<br>デフォルトでは、カレントディレクトリにあるクレートのみが対象になる。 |
| --check | フォーマットは適用せず、チェックのみを行う。                                                                  |
