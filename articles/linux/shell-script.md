## arguments

| what      | description                                  |
| --------- | -------------------------------------------- |
| $0        | script name                                  |
| $1        | 1番目の引数                                  |
| $@        | すべての引数                                 |
| ${1:-foo} | 1番目の引数のデフォルト値としてfooを代入する |
